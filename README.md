# C++ Learning Roadmap

This repository contains a structured roadmap for learning C++ from **absolute beginner** to **advanced** levels. It is designed to help you progressively build your knowledge and experience with C++ while pushing your progress to GitHub. Each section includes key topics and suggested tasks to reinforce learning through implementation.

---

## Getting Started

### Prerequisites
- Install a C++ compiler (GCC, Clang, or MSVC).
- Set up an IDE or text editor (Visual Studio, CLion, VS Code, etc.).
- Install Git and create a GitHub account.

### Initial Setup
1. Clone this repository:
   ```sh
   git clone https://github.com/YeghiazarianVahe/CPlusPlus
   cd CPlusPlus
   ```
2. Create a new branch for your work:
   ```sh
   git checkout -b feature-initial-setup
   ```
3. Start coding and commit your progress frequently.

## Roadmap Overview

### 1. Setting Up Your Environment
- Install and configure a C++ compiler.
- Learn basic command-line compilation.
- Initialize a Git repository and push your first commit.

### 2. Basic C++ Syntax
- Variables, data types, and operators.
- Basic input/output (`std::cin`, `std::cout`).
- Implement simple programs (Calculator, Temperature Converter).

### 3. Control Flow
- Conditional statements and loops.
- Implement a Number Guessing Game.

### 4. Functions
- Function definitions, overloading, and recursion.
- Implement Factorial and Fibonacci calculations.

### 5. Arrays, Strings, and Vectors
- Work with static arrays and `std::vector`.
- Implement array statistics and string reversal functions.

### 6. Pointers and References
- Learn about pointers, memory allocation, and references.
- Implement a simple dynamic array class.

### 7. Object-Oriented Programming (OOP)
- Learn about classes, objects, and constructors.
- Implement a `Circle` and `BankAccount` class.

### 8. Inheritance and Polymorphism
- Understand base and derived classes, virtual functions.
- Implement Shape and Employee class hierarchies.

### 9. Operator Overloading
- Overload arithmetic and comparison operators.
- Implement a `Complex` number class.

### 10. Memory Management and RAII
- Learn about smart pointers (`std::unique_ptr`, `std::shared_ptr`).
- Implement RAII principles in a class managing file handles.

### 11. Templates and Generic Programming
- Create function and class templates.
- Implement a generic container class.

### 12. Standard Template Library (STL)
- Work with STL containers (`std::vector`, `std::map`).
- Implement sorting and searching algorithms.

### 13. Exception Handling
- Use `try`, `catch`, and `throw`.
- Implement exception handling in a Bank Account system.

### 14. File I/O
- Read and write files using `std::ifstream` and `std::ofstream`.
- Implement a text file logger.

### 15. Concurrency (C++11 and beyond)
- Work with threads and synchronization primitives.
- Implement a multithreaded counter.

### 16. Advanced C++ Features & Best Practices
- Explore C++17/C++20 features like structured bindings and coroutines.
- Implement a modern C++ features showcase.

### 17. Design Patterns in C++
- Learn Singleton, Factory, Observer, and Strategy patterns.
- Implement a strategy-based sorting system.

### 18. Final Project & Portfolio
- Develop a CLI-based project (e.g., To-Do List Manager).
- Build a small game or simulation.
- Work on a networking/multithreading project.

## How to Contribute
1. Fork the repository.
2. Create a new branch:
   ```sh
   git checkout -b feature-your-feature-name
   ```
3. Commit your changes:
   ```sh
   git commit -m "Added new feature"
   ```
4. Push to your forked repo:
   ```sh
   git push origin feature-your-feature-name
   ```
5. Create a pull request.

---

## Best Practices for Learning
- **Commit Often:** Push code regularly to track progress.
- **Use Branches:** Work on features separately before merging.
- **Document Your Code:** Write comments and explanations.
- **Keep Learning:** Follow modern C++ standards and best practices.

By following this roadmap and completing the tasks, you'll develop a strong foundation in C++ and build a valuable portfolio on GitHub.

**Happy coding!** 🚀
